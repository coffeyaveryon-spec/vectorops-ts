# vectorops-ts

**Status: In Progress**

TypeScript/Node.js port of VectorOps — a production-style RAG retrieval platform.

## Tech Stack
- TypeScript / Node.js
- Express.js
- LangChain
- Vector store integration
- REST API

## What it does
Document ingestion and semantic search system that lets users query their own documents conversationally using RAG pipelines and LLM integration.
